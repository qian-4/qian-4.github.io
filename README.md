タイトル：CRIMSON PROTOCOL（クリムゾン・プロトコル）

ネオ・マリブ区、午後21時。
冷たい酸性雨が、壊れたネオン看板を濡らしていた。
カスタマー・コア社の受付フロアでは、AI受付端末が淡々と光を放つ。

ジェシカは強化ガラス越しに外を見ながら、今日も長いシフトを耐えていた。
「また停電かよ…」
天井の蛍光管が瞬き、フロアの照明が一瞬だけ赤に揺れる。

その瞬間だった。

金属ドアが音を立てて叩き割られ、
人影がひとつ、フロアに侵入した。

「おい、俺のアカウント凍結の理由を説明しろよ。
──説明しねぇなら、誰かひとり消すぞ。」

男の右腕は、明らかに非合法サイバー義肢。
先端は刃物とも工具ともつかない形状に変形し、
青い電流がバチバチと走っていた。

🔴 1. プロトコル CRIMSON 発動（＝即時退避）

ジェシカの視界に、ARレンズが赤い警告を表示する。
《CRIMSON PROTOCOL 発動 — 即時退避推奨》

彼女はためらわなかった。
椅子を蹴って後方の緊急退避ルートへ走る。

「マーク！セクション3経由でバックヤードへ！」
同僚たちもAIの誘導に従い、出口に向かって走る。

後方で、男の義肢が床を焼きながら引きずられる音がした。

🔴 2. 企業ネットに通報 — 警備部隊出動要請

退避しながらジェシカは腕のホロパネルを開く。
《警備部隊（CorpSec）に通報しますか？》

「即時。致死性武装の可能性あり。」

数秒で、企業治安部隊《CorpSec》から返信が入る。
《了解。ドローン班を先行投入。8分で地上部隊到着》

天井のスリットが開き、小型ドローンが次々と飛び出した。

🔴 3. フロア封鎖と隔離

マークが背後で叫ぶ。
「セクション4、オートロック作動！
侵入者をフロアに閉じ込める！」

金属シャッターが自動で降り、侵入者の退路を塞ぐ。
男が怒号を上げてシャッターを殴るたび、火花が散った。

🔴 4. 証拠ログの確保

AI管理システム《SHELTER》が自動で動き出す。

《監視映像バックアップ完了》
《音声データ確保》
《侵入者の義肢シリアル追跡開始》
《心理ケアプログラム起動》

ジェシカの端末に、深呼吸を促す青いガイドが浮かんだ。

🔴 5. 心理ケアと保護

避難室に逃げ込んだジェシカは、
震える手を握りしめながら椅子に座り込む。

外では、ドローンの旋回音と企業治安部隊のブーツ音が響く。
彼女は息を整えながら、
いつもの言葉を自分に繰り返した。

「CRITICALは、私が戦う領域じゃない。
生き延びる。それがプロトコルの第一条…」

防護ドアの向こうで、銃声とも金属音ともつかない衝撃音が響く。

やがて、通信が入る。

《現場制圧完了。従業員の安全を確認中》

ジェシカは静かに瞼を閉じた。
この都市では、生き延びるだけで十分強い。
